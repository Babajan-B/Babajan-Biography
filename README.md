Personal single-page site for Dr Babajan Banaganapalli.

Tech stack: Next.js App Router, TypeScript, Tailwind CSS, Framer Motion.

Getting started
```bash
npm install
npm run dev
```

Editing content
- Hero and sections live in `app/page.tsx` with clear structure.
- Replace `public/profile.jpg` with your photo. If missing, a placeholder is shown.
- Update contact links and email in the Contact section.
- Add real items to Work/Portfolio/Blog where marked.

API
- Minimal `/api/contact` POST logs the payload. No secrets required.

Deploy
- Push to GitHub and import on Vercel. Defaults are fine.
